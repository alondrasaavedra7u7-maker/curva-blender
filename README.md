# Curva en Blender con Python

## Descripción

Este proyecto contiene un script en Python para Blender que permite colocar una fila de objetos siguiendo una curva.

El script automatiza la generación y posicionamiento de objetos a lo largo de una trayectoria curva.

---

## Tecnologías utilizadas

- Blender 5.0
- Python (Blender Scripting)

---

## Cómo usar el script

1. Abrir Blender.
2. Ir a la pestaña **Scripting**.
3. Crear un nuevo script.
4. Copiar el contenido de `curva_blender.py`.
5. Ejecutar con **Run Script**.
